# currency-site
# موقع أسعار العملات  موقع بسيط يعرض أسعار العملات مقابل الدولار الأمريكي باستخدام واجهات برمجية مجانية (Frankfurter و Exchangerate.host).   يحتوي أيضًا على محول عملات مباشر يمكن من خلاله تحويل أي مبلغ بين عملتين مختلفتين.   تم تصميم الموقع باستخدام HTML و CSS و JavaScript ليكون سريعًا وسهل الاستخدام.
